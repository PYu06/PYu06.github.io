The Enumerable#Map
Oct 12 '14

The Enumerable#map method takes the form: array.map{ |item| block }. Calling this method will create a new array that will do the code block to each item in the array.

A good time to use the .map method is when you want to create a new array after doing a block of code to the original array. Think of it as iterating each element and outputting into a new array.

For example: 


	[1,2,3,4].map { |x| x*2 }
	# => [2,4,6,8]
