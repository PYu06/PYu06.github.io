Variable Scopes
Oct 28 '14

As a reminder: variables in Ruby are declared by defining a variable name = to a value. e.g. some_string = "Hello World!". Thus a scope of a variable is where the variable is visible in a Ruby program. Ruby has 5 variable scopes: a global variable, a class variable, an instance variable, a local variable, and a constant.

A global variable is preceeded by a $. I.e: $global_var = "Everyone can see me =O!". As the value of $global_var states, a global variable is accessible from anywhere in the Ruby program. It doesn't even matter where you put the global variable in your program. For this reason, it is advised that you don't use global variables as they can also be changed from anywhere in the program making it hard to track any bugs that may occur.

A class variable is defined by: @@class_var = "Hello all instances!", and it has to be defined before you can call it (unlike global variable which you can place anywhere in the program). Any instance within the class can access and alter the class variable. If you change the class variable in one method, it changes the value of the variable for all the other methods you call on it.

An instance variable has the following prefix: @instance_var = "I'm a little more selective". Instance variables are similar to class variables, except for the fact that the value only changes in the method that is called in; it doesn't change the value of @instance_var for the whole class.

A local variable doesn't have a prefix, and are usually all lower case: local_var = "I'm quite short lived =(". It can only be accessed and called within the method that it is defined in. A common use for a local variable is when you want to use a counter method.

Constants are as their name suggests, a variable that (should) stays constant throughout the program. They can be defined anywhere within a class. If you do decide to change the value of a constant, Ruby will give you an error saying that the value of the constant has changed. Oh, and constants look like this: CONSTANT = "I ABHOR CHANGE!"
