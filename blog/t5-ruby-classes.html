Ruby's Got Class
Oct 17 '14

Greetings reader! Today I'm going to talk about Ruby Classes. A class is like a title given to contain a group of objects that you want to act the same way. I'm going to create an example class for habits. To define a class you would do:

class Habit
	end

Within a class, you can instance methods and instance variables.Instance methods and variables are basically the same as methods and variables except for the fact that they are shared by all objects inside the class, and are preceeded by an "@" sign. 

class Habit
		def initialize(name, start_date)
			@name = name
			@start_date = start_date
		end

		def name
			@name
		end

		def start_date
			@start_date
		end

		def stop_date
			p "You should do #{@name} for at least 21 days starting from #{@start_date}. Work hard!"
		end
	end

With the previous code block, you can now create a new habit by typing:

 ex = Habit.new("Exercise","Oct 17 '14")

You can then call the method stop_date:

ex.stop_date

Which will output:

"You should do Exercise for at least 21 days starting from Oct 17 '14. Work hard!"

You can now create as many instances as you want for class Habit, and they will be able to do all the method calls you define within the class.
