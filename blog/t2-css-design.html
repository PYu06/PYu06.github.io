The Box Model

The easiest way to explain the difference between Margin, Border, and Padding for CSS is to use The Box Model:

As you can see from the image, the margin is the outermost layer of the box. It is an invisible divider between this object and other objects around it. There are several ways you can format margin:

p {
	margin-top: #px;
	margin-bottom: #px;
	margin-right: #px;
	margin-left: #px;
}

Or you can use a short form version:

p {
	margin: #px #px #px #px;
}

The order goes from top, right, bottom, to left.
You could also do:

p {
	margin: 10px 30px 50px;
}

This specifies that the top will have a margin of 10px, the right and left will have a margin of 30px, and the bottom will have a margin of 50px.
If you only specify two margins, then the top and bottom will have the same margin of 20px, and the right and left will share the same margin of 30px:

p {
	margin: 20px 30px;
}

Lastly, you can specify the margin for all four sides of the box by:

p {
	margin: 35px;
}

Border is what you think it is. A border around your object. You can control the style, width, and colour. The different styles of borders you can have are: none, dotted, dashed, solid, double, groove, ridge, inset, and outset. The width is determined by pixels. The colour can be changed using either a name, a RGB value, or a hex value.
Like the margin property, you can format the border in several ways: 

p {
	border-style: solid;
	border-color: black;
}

Or: 

p {
	border-top-style: double;
	border-right-style: dotted;
	border-bottom-style: ridge;
	border-left-style: solid;
}

Just like with margin, you can also use a short hand form:

p {
	border-style: double dotted ridge solid;
}

Double top, dotted right and left, ridge bottom:

p { border-style: double dotted ridge; }

Double top and bottom, dotted right and left:

p {
	border-style: double dotted;
}

All four solid:

p {
	border-style: solid;
}

You can also combine width, style, and colour into one line:

p {
	border: 3px dotted red;
}

N.B. You need to select a "border-style" in order for "border-color" to work.

Finally padding! Padding is the space between the content and the border. Just like with margin and border, you can format padding in multiple ways: individually, top right bottom left, top right/left bottom, top/bottom right/left, and all four at once.
The shorthand format is:

p {
	padding: #px;
}

To summarise, the difference between Margin, Border, and Padding is that they are different properties of The Box Model. Margin is the transparent outermost layer, border is the outline around the object, and the padding is the area that separates the content from the border.
